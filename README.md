# Password Strength Checker 🔐

A simple Password Strength Checker built using HTML, CSS, JavaScript, and Python.

This project checks whether a password is strong or weak based on basic security rules.

## Features

- Checks minimum password length
- Checks for at least one number
- Checks for at least one uppercase letter
- Checks for at least one special character
- Interactive web interface
- Python CLI version included

## Technologies Used

- HTML
- CSS
- JavaScript
- Python

```bash
Password-Strength-Checker/

```

## Password Rules

A strong password must have:

- Minimum 8 characters
- At least 1 number
- At least 1 uppercase letter
- At least 1 special character

Example:

```text
Password@123
```

## How to Run (Web Version)

1. Clone the repository

```bash
git clone <repository-link>
```

2. Open the project folder

```bash
cd Password-Strength-Checker
```

3. Open index.html in browser

OR

Use Live Server in VS Code.

## How to Run (Python Version)

Run:

```bash
python pro3.py
```

Enter password and check strength.

## Output Example

Strong Password:

```text
✅ Strong Password
```

Weak Password:

```text
❌ Weak Password
```
